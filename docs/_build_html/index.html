PEXTANT 0.1¶

PEXTANT, short for python-SEXTANT, is a tool for astronaut EVA path planning on planetary surfaces. It is intended to be integrated with the MINERVA suite for the BASALT project and is based off of SEXTANT from MIT’s Man Vehicle Laboratory.
